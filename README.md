## 시작하기

### 의존성 설치

본 서비스를 실행하기 위해선 의존성 외부 모듈들을 설치해야만 합니다.<br>
아래의 명령어를 통해 의존성 모듈을 설치해주세요.
<pre>
$ npm install
</pre>

### 서비스 실행

`TypeScript` 언어로 작성된 서비스 애플리케이션이 `ts-node`를 통하여 별도의 트랜스 컴파일링 없이 실행됩니다.<br>

<pre>
$ npm start
</pre>
